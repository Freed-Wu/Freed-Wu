# my-dotfiles

Keep all my computers' configurations same.

`~/.config/nvim` is not included because it is too heavy.

- Linux
  - GNU/Linux
    - Gentoo
    - Archlinux (PC)
    - Deepin (PC)
    - Openwrt (router)
    - Ubuntu (docker)
  - Android
    - Termux (mobile)
- Windows
  - Cygwin (U disk)
  - Msys (U disk)
